Антон Наумов

Разрабатывает комплексные Java-решения с 1999 года. За это время успел поработать в множестве больших и маленьких компаний Харькова. Работал в должности Java Developer, Team Lead, Project Manager, Head of Java Department.

Работал с использованием как классических, так и гибких практик управления проектами. Решал самые различные задачи - от простейших прототипов до сложных распределенных систем - в самых различных областях (Telecommuncations, Billing, JSF components, Liferay, Pentaho BI).

Сейчас работает на должности лидера команды Java разработки в зарубежном стартапе.

Блог: corwin.calepin.co
